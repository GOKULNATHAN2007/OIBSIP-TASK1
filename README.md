Task 1 - LANDING PAGE FOODMUNCH
– Responsive Food Store Website
Tagline: Get Delicious Food at Anytime.

 => OBJECTIVE 

The objective of this project is to design and develop a modern, responsive, and user-friendly food store landing page that provides an engaging experience for visitors. The website showcases food categories, healthy food options, delivery services, payment information, customer offers, and social media links while ensuring compatibility across different devices. This project also aims to improve frontend web development skills using HTML5, Tailwind CSS, and CSS.

=> STEPS PERFORMED 

- Planned the website layout and organized the content into multiple sections.
- Created the webpage structure using HTML5.
- Styled the website using Tailwind CSS along with custom CSS for additional design improvements.
- Implemented a responsive navigation and hero banner.
- Developed the "Why Choose Us" section to highlight the website's services.
- Designed the "Explore Menu" section to display different food categories.
- Added a Healthy Food section with attractive images and descriptions.
- Included Delivery & Payment information for customer convenience.
- Created a Customer Gift section to promote special offers.
- Added a Follow Us section with social media icons and a professional footer.
- Tested the website on different screen sizes to ensure responsiveness and a smooth user experience.

 => TOOLS USED

- HTML5
- Tailwind CSS
- CSS3
- Google Fonts (Roboto)
- Font Awesome Icons
- Visual Studio Code
- Git & GitHub

 => OUTCOMES

The FoodMunch project was successfully completed as a responsive food store landing page with a clean and attractive user interface. The website adapts well to desktop, tablet, and mobile devices, providing a consistent user experience. Through this project, I strengthened my knowledge of HTML5, Tailwind CSS, responsive web design, layout creation, styling techniques, and frontend development best practices. It also enhanced my confidence in building professional-looking websites and managing projects using GitHub.
